# Comf-i


## 👗 What it is
**Comf i Virtual Wardrobe** is a personal web application that allows users to digitally organize their clothing collection and build outfits based on categories and different weather conditions.

## 🛠️ What it does
* **Previews Clothes:** Displays your clothing pieces using a gallery grid that switches to a sketch view when you hover over them.
* **Filters Wardrobe:** Lets you narrow down your items by clothing category and weather matching.
* **Styles :** Allows you to click items to overlay and layer them dynamically without mixing.
